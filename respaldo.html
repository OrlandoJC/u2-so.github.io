Respaldo

Pasos de instalación

Paso 1

En configuración de virtual box, encontramos la opción de “Carpetas compartidas” el cual tenemos previamente creado en Windows 10 con el nombre “Docs” y de esa manera poder utilizarlo en nuestra máquina virtual

Ver imagen

Paso 2

Dentro la carpeta Documentos de Windows server creamos un archivo llamado “Script” con la extensión “.bat” el cual al ejecutarlo nos genera un archivo llamado “Hola mundo”.

Ver imagen

Paso 3

Siempre dentro la carpeta Documentos en Windows server creamos un archivo llamado “respaldo” con la extensión “.bat” el cual al ejecutarlo nos genera un archivo “dat.txt” el cual nos dice que ya tenemos guardado un respaldo de nuestros archivos “.txt” en la carpeta “Docs”.

Ver imagen

Paso 4

Podemos visualizar la carpeta “Docs” en windows server ya con los scripts ejecutados.

Ver imagen

Paso 5

En la barra de búsqueda de Windows Server encontramos “Programador de tareas” en la opción de crear nueva tarea en “Biblioteca de programador” y se configura con el nombre “Respaldo-Archivos” y seleccionamos las opciones “ejecutar solo cuando el usuario haya iniciado sesión” y “Ejecutar con los privilegios más altos”.

Ver imagen

Paso 6

En la pestaña de desencadenadores le damos click a nuevo y nos saldrá una ventana en donde configuraremos cuando queremos que se respalden los archivos, ponemos nuestras configuraciones y le damos click a aceptar.

Ver imagen

Paso 7

Se muestra la pestaña de “Crear tarea” en la sub-opcion “Desencadenadores” con uno ya programado.

Ver imagen

Paso 8

Se muestra la pestaña general de “Programador de tareas” ya con la tarea activa de “Respaldo-Archivos”.

Ver imagen

Paso 9

Se muestra el archivo “dat.txt” momentos después de haberse ejecutado la tarea “Respaldo-Archivos” en la carpeta compartida “Docs”.

Ver imagen

Paso 10

Se muestra el contenido del archivo “dat.txt” para verificar que el respaldo se realizó correctamente.

Ver imagen

Paso 11

Se muestra el contenido de la carpeta compartida “Docs” de manera local en Windows 10.
